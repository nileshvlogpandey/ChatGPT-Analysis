# ChatGPT Analysis

## 📊 Project Overview
This project focuses on analyzing user feedback and reviews for ChatGPT. The objective is to evaluate user satisfaction, detect major functional boundaries or issues, and observe how user sentiments change over time.

## 📋 Dataset Structure
The dataset contains 4 key fields:
* **Review ID:** A unique identifier for every review.
* **Review:** Text comments and feedback given by the user.
* **Rating:** Level of satisfaction represented on a scale of 0-5.
* **Review date:** The timestamp when the review was posted.

## 🛠️ Key Project Objectives
1. **Sentiment Analysis:** Categorizing user satisfaction levels into Positive, Negative, and Neutral based on Ratings.
2. **Issue Identification:** Analyzing negative reviews to pinpoint common bugs, errors, or user frustrations.
3. **Time-Series Analysis:** Tracking the chronological evolution of feedback to detect spikes or trend shifts over time.

## 💻 Tech Stack
* **Language:** Python
* **Environment:** Google Colab / Jupyter Notebook
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
